# FreenomBot
The bot will auto-renew domain name
---
# How to use it

## Edit config.toml
Please type Freenom account(s)
``` toml
[[Accounts]]
Username = "xxx@xxx.com"
Password = "xxx"

[[Accounts]]
Username = "ooo@ooo.com"
Password = "ooo"
```

## Launch FreenomBot

``` sh
./FreenomBot
```
